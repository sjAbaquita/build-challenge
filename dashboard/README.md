# CareRoster — Rostering Dashboard UI

A responsive, static SaaS dashboard UI for a fictional hospital rostering system. This project focuses on real-world product UI design, information hierarchy, and usability for shift scheduling and workforce management.

Built using HTML, Tailwind CSS, and vanilla JavaScript.

---

## Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript

---

## Project Overview

This project simulates a real hospital rostering application used by administrators to manage staff schedules, view weekly shifts, handle shift details, and communicate with staff.

The main focus is on building a dense but readable enterprise-grade UI with strong hierarchy and realistic SaaS patterns.

---

## Key Features / Screens

- Sidebar navigation (multi-section app layout)
- Weekly rostering grid view (core screen)
- Shift detail side panel
- Staff and department sections (mock UI screens)
- Shift requests management UI
- Reports / analytics dashboard (static mock layout)
- Internal messaging / chat interface
- Responsive mobile navigation (drawer style)
