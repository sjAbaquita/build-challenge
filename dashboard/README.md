# CareRoster — Rostering Dashboard UI

A responsive, static SaaS dashboard UI for a fictional hospital rostering system.

Built using HTML, Tailwind CSS, and vanilla JavaScript.

---

## Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript

---

## Project Overview

This project is a UI concept for a hospital rostering SaaS dashboard focused on the weekly scheduling view. The goal was to design a clean and realistic interface where admins can manage staff schedules, view shifts across the week, and quickly access shift details.

The dashboard includes a sidebar navigation, top filter bar, weekly roster grid, and a shift detail panel using mock data only. I explored multiple layout directions first, then refined the strongest version into a more polished and production-style SaaS interface inspired by modern healthcare scheduling systems.
