# CareRoster Landing Page

A modern, mobile-first SaaS marketing landing page for a fictional hospital rostering platform. Built using plain HTML, Tailwind CSS, and vanilla JavaScript.

---

## Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript

---

## Features

- Mobile-first responsive design
- Sticky navigation with mobile menu
- Hero section with CTA
- Features section
- Product showcase/mockup section
- Pricing / workflow section
- FAQ accordion
- Final CTA + footer
- Smooth animations and transitions

---

## Running Locally

Since this project uses plain HTML/CSS/JS, no build step is required.

### Open Directly
Simply open `index.html` in your browser.
